# CiaLli

<img align='right' src='public/assets/home/default-logo.png' width='180px' alt="CiaLli logo">

CiaLli 是一个现代化、功能丰富的全栈社交站点（前端），基于 [Mizuki](https://github.com/matsuzaka-yuki/Mizuki) 模板开发。使用 [Directus](https://directus.io/) 作为无头 CMS 后端，负责数据存储、用户管理和文件托管。

> [!NOTE]
> 本项目大部分代码、测试和文档均由智能体编写，人工仅对功能进行简单测试。
> 项目作者对代码安全与质量不作任何负责。
